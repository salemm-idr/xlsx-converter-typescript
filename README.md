###how to use?

1. Clone git repo
----
    git clone https://github.com/salemm-idr/xlsx-converter-typescript.git


2. Install with
----
    npm install


3. Set the env
----
    create .env and set PORT

4. Run with 
----
    npm run build 
    npm run dev
